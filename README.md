# Great Sage Project

A simple visual novel style game built with Love2D.

## Development

- Engine: [Love2D](https://love2d.org/)
- Web Export: [love.js](https://github.com/Davidobot/love.js)

## How to Play

### Desktop
Install Love2D and run:
```bash
love src/
```

### Web
(Coming soon via love.js)

## Project Structure
- `src/`: Game source code (Lua)
- `assets/`: Images, sounds, and music (planned)
